05 — Guía de uso
Correcto
El ámbar es el color protagonista: CTA principal, cifras destacadas, nombre de la marca.
Usar el negro como base dominante — la pantalla "respira" oscuridad.
Verde exclusivamente para estados positivos: pagado, stock OK, confirmación.
Cormorant Garamond para la marca y los nombres de producto únicamente.
DM Mono para toda la interfaz funcional: etiquetas, cifras, botones.
El botón primario lleva texto en negro oscuro sobre fondo ámbar — nunca blanco.
Incorrecto
No usar amarillo brillante o neón — el ámbar debe evocar oro envejecido, no señalización.
No combinar ámbar y verde en el mismo elemento — son colores de jerarquías distintas.
No usar gradientes, glows ni efectos de brillo sobre el ámbar.
No usar el ámbar como fondo de pantallas completas — reservarlo a elementos puntuales.
No mezclar con magenta o colores fríos — rompen la temperatura cálida del sistema.
No usar tipografías sans-serif genéricas (Inter, Roboto) — destruyen el carácter editorial.
